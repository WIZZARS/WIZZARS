| Category | Details |
|---|---|
| Stack | React Native, TypeScript, Node.js, Supabase, PostgreSQL |
| Scale | Multi-clinic cross-platform architecture |
| Performance | 60% reduction in manual record-keeping time |
| Security | Row-Level Security, RBAC, Secure REST APIs |
